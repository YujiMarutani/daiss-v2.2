## Overview
DAISS v2.2 is a minimal, self‑starting operating system for sovereign intelligence.  
It models intelligence as a recursive spiral integrating the Black Box Problem, the Ternary Epistemic Engine, PDSCA/Disco‑GenTabs, and the Bio‑Digital Twin.  
This frozen, DOI‑anchored release adds a unified YAML node schema and a reproducible Markdown‑to‑PDF workflow.

## Spiral Meta‑Structure
DAISS defines intelligence as a regenerative spiral:
Black Box → Recognition → Standardization → Bio‑Digital Integration → New Black Box.  
See the PDF for full diagrams.

## Ternary Epistemic Engine
DAISS operates through a three‑state loop:
- **‑1 Purge** — pathology detection  
- **0 Equilibrium** — homeostasis  
- **+1 Amplify** — sovereign growth  

This loop forms the epistemic kernel of the system.
